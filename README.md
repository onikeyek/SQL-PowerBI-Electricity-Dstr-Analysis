# ⚡ Electricity Distribution Analysis — SQL + Power BI

End-to-end data analysis project on electricity distribution data — from raw SQL queries to interactive Power BI dashboards that surface operational insights for decision-makers.

---

## 📊 What This Project Does

Takes raw electricity distribution data and transforms it into actionable business intelligence:

- **SQL (T-SQL)** — data extraction, cleaning, and aggregation
- **Power BI** — interactive dashboards for non-technical stakeholders
- Identifies distribution patterns, anomalies, and efficiency metrics

---

## 🛠️ Tech Stack

`T-SQL` · `Microsoft SQL Server` · `Power BI`

---

## 📁 Project Structure

```
├── queries/          # T-SQL scripts for data extraction & transformation
├── dashboard/        # Power BI report file (.pbix)
└── README.md
```

---

## 🔍 Key Analyses

- Distribution volume trends over time
- Regional consumption breakdowns
- Anomaly detection in usage patterns
- Summary metrics for operational reporting

---

## ⚡ How to Use

1. Run the SQL scripts in `queries/` against your SQL Server instance
2. Open the `.pbix` file in Power BI Desktop
3. Connect to your data source or use the sample dataset

---

## 💡 Why This Project

Built to demonstrate the full analyst workflow: writing production-quality SQL, transforming data into clear visuals, and communicating findings to non-technical audiences — the same skill set needed when redesigning how an organisation operates with data.
